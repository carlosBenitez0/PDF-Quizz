INDICACIONES DE USO

Variables necesarias para la generación del quizz

Version de GPT: Esta variable indica la versión del modelo de GPT que se está utilizando.

Configuración: Define un rango que representa las páginas que se tomarán en cuenta del PDF, y también, la cantidad de preguntas que se generarán a partir del PDF cargado.

  • Se debe respetar estrictamente el siguiente formato:inicio-final,cantidad .
  • Inicio: Indica la primera página que se tomará del documento PDF. Es el inicio del rango.
  • Final: Indica la última página que se tomará del documento PDF. Es el final del rango.
  • Cantidad: Cantidad total de preguntas a generar de todo el documento pdf cargado.

Estructura: Define la estructura o el formato del cuestionario generado.

  • Términos y Definiciones: El cuestionario contiene pares de términos y sus definiciones.
  • Estándar: El cuestionario sigue un formato estándar predefinido.
  • Ambas: El cuestionario incluye tanto términos y definiciones como el formato estándar.

Creatividad: Establece el nivel de creatividad que se le permite al modelo a la hora de generar sus resultados. Mientras más baja sea, más apegado al texto serán las preguntas y sus posibles respuestas.

  • Baja: La creatividad del modelo es baja.
  • Media: Las creatividad del modelo es intermedia.
  • Alta: La creatividad del modelo es alta.

Importante: No hay un límite para el tamaño del PDF: se crean bloques de texto de una cantidad similar de palabras (menores a 1024 palabras). Pero mientras más grande sea, más tiempo se tardará en generar el quizz debido a que la cantidad de bloques de texto generados desde el documento es mayor. Para cada bloque de texto se realiza una petición de determinada cantidad de preguntas. Cuando se terminan de trabajar todos los bloques de texto se muestra el resultado en la parte inferior de la página principal.
