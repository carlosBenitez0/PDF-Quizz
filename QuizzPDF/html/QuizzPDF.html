INDICACIONES DE USO

Variables necesarias para la generación del quizz

Version de GPT: Esta variable indica la versión del modelo de GPT que se está utilizando.

Cantidad a generar: Define la cantidad de preguntas que se generarán a partir del PDF cargado.

Orden: Determina el orden en el que se presentarán las preguntas generadas.

  • Ordenado: Las preguntas se presentan en el orden en que aparecen en el PDF.
  • Aleatorio: Las preguntas se presentan en un orden aleatorio.

Estructura: Define la estructura o el formato del cuestionario generado.

  • Términos y Definiciones: El cuestionario contiene pares de términos y sus definiciones.
  • Estándar: El cuestionario sigue un formato estándar predefinido.
  • Ambas: El cuestionario incluye tanto términos y definiciones como el formato estándar.

Dificultad: Establece el nivel de dificultad de las preguntas generadas.

  • Baja: Las preguntas son de un nivel de dificultad básico.
  • Media: Las preguntas tienen un nivel de dificultad intermedio.
  • Alta: Las preguntas son de un nivel de dificultad avanzado.
